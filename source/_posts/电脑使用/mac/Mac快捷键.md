# 通用

```
复制： Command + c
剪切： Command + x
粘贴： Command + v
撤销： Command + z
反撤销： Command + Shift + z
全选： Command + a
查找： Command + f
查找下/上一项： Command + g/shift + g

打开表情键盘： ctrl + Command + 空格
打开聚焦搜索： Command + 空格

隐藏最前面的应用窗口：Command + h
最小化最前面的应用至程序坞：Command + m
最小化最前面的应用的所有窗口至程序坞：Command + Option + m

打开： Command + o
打印： Command + p
存储： Command + s
打开新标签页： Command + t
关闭最前面的窗口： Command + w
关闭应用的所有窗口： Command + Option + w
强退应用： Command + Option + Esc
```

# 文本编辑

## 移动光标

```
以字符为单位移动:  方向键←/→
移动到上/下一行相同水平位置：方向键↑/↓
以词语(中英)为单位移动:  Option + 方向键←/→
以行为单位移动:  Command + 方向键←/→
以段落为单位移动:  Option + 方向键↑/↓
以整个文档为单位移动:  Command + 方向键↑/↓
 
```

## 滚动

```
翻页(Page Up/Down): fn + 方向键↑/↓
滚动至文稿开头/末尾（Home/End）：fn + 方向键←/→

```

## 选择文本

```
移动光标的快捷 + Shift
滚动的快捷 + Shift
```

## 编辑

```
向前删除：delete  
向后删除：fn + delete
删除至段尾：ctr + k
光标处换行：ctr + o
光标前后两个字符交换：ctr + t

```

# 文件

```
打开一个新的“访达”窗口：Command + n
打开设置窗口： Command + 逗号
新建文件夹：Command + Shift + n

快速预览所选文件：空格（或Command + Y）
幻灯片全屏快速查看所选文件：Option + 空格（或Command + Y）

列表视图进入上/下一层级右箭头：方向键←/→

复制粘贴： Command + c 然后 Command + v
剪切粘贴： Command + c 然后 Command + Option + v
当前目录复制粘贴：Command + d

推出磁盘 Command + e
访达中搜索文件 Command + f
显示文件简介 Command + i
显示别名对应的真正文件 Command + r

打开应用程序文件夹 Command + Shift + a
打开电脑文件夹 Command + Shift + c 
打开桌面文件夹 Command + Shift + d
打开最近使用文件夹 Command + Shift + f
打开前往文件夹窗口 Command + Shift + g  
打开当前用户文件夹 Command + Shift + h 
打开icloud文件夹 Command + Shift + i
打开网络窗口 Command + Shift + k
打开下载文件夹 Command + Shift + l

将所选项移到废纸篓：Command + Delete
将所选项删除(不经过废纸篓)：Command + Option + Delete
将所选项删除(不经过废纸篓)而不显示确认对话框：Command + Option + Shift + Delete
```

# Visual Studio

无与伦比的 IDE, Community版免费, 其余付费

总结: 非常智能, 拥有强大的功能, 可以完成基本上所有 C++ IDE 无法完成的任务

优点: 支持远程合作 (Live Share), 智能化的代码补全, 具有很强的稳定性, 可以进行扩展与自定义, 完全支持中文

缺点: 编译运行速度慢; 对程序的要求十分严格; 不适合初学者

## 安装

### 在线安装

官网下载，双击安装即可在线安装

下载速度还是相对较慢的，900多MB

### 离线安装

Mono Framework，Visual Studio for Mac 顺序安装即可

# Visual Studio Code
